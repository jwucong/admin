# admin

### 1. 环境要求
   - **node >= 8.3.0**
   - **npm  >= 5.3.0**
   
### 2. 开发/构建/分析
   1. **开发**
   
      ```shell
        npm start
        // or
        npm run dev
      ```
      
   2. **格式化代码**
      ```shell
         npm run format
      ```

   3. **构建**
   
      ```shell
        npm run build
      ```

   4. **分析**
      ```shell
        npm run analysis
      ```
